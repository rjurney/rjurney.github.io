Thanks to Joe Crobak for the tip, I have patched the Python library for Avro to build under Mac OS X by removing its python-snappy dependency. This has been bugging a lot of people, and while the patch removes snappy compression… I have no idea what that is, and I don’t need it :)

The patch is available here: AVRO-981.

To build Avro, you’ll need to clone trunk from https://github.com/apache/avro.git, apply the patch and then build and install the python library.

Update: it seems the problem was that Snappy is a dependency. Install it from here: http://code.google.com/p/snappy/ and python-snappy and python avro should work :)
