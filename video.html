Realtime Predictive Analytics

Russell Jurney
Agile Data Science
Full Stack Analytics Applications with Python, Kafka and Spark

This is a professional development class that teaches how to iteratively craft entire analytics web applications using Python, Flask, PySpark, Kafka, MongoDB, ElasticSearch, Bootstrap and D3. This stack is a popular one and is an example of the kind of stack needed to process and refine data at scale in real world applications of data science. During this course, students will use airline flight data to produce an entire analytic web application, from the ground up.
